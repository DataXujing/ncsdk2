    API-Only Installation for Windows

    This feature is only available with Intel® Movidius™ Neural Compute SDK 2.x.

    The Intel® Movidius™ Neural Compute API v2 (Intel® Movidius™ NCAPI v2) can be installed on Windows, enabling you to develop and run NCAPI v2 applications in a Windows environment. This installation is NCAPI-only and will not install any Intel® Movidius™ Neural Compute SDK tools. Network graphs will need to be compiled on a supported host machine or virtual machine.

    Prerequisites

    • Development computer with 64-bit Windows 10 (validated on 10 but may work on other 64-bit editions of Windows)
    • The ncsdk GitHub repo or .tar.gz release package
    • Optional: Python with numpy, OpenCV

    Installation

    Install the NCAPI in Windows by performing the following steps:

    1. If you want to install the Python API, install a supported version of Python (3.5 or 2.7) and the numpy module (>=1.13.0,<=1.13.3) before running the NCAPI installer. Make sure that python.exe is in your Path environment variable.
    2. Clone the ncsdk GitHub repository to your Windows machine, or if you received the release in a .tar.gz archive copy it to your Windows machine and decompress it (7zip or WinZip can do this).
    3. Navigate to the ncsdk/win_api_install directory and run the installer appropriate for development of your application (WIN32_setup.exe for 32-bit applications or WIN64_setup.exe for 64-bit applications). If you have a previous version of the NCAPI installed, you will be prompted to uninstall first.
    4. Choose an installation location or accept the default location.
    5. If you are installing the Python API, click Yes when prompted. If you choose not to install the Python API, you can install it later by re-running the NCAPI installer.

    Examples

    Pre-built example

    One pre-built example is located in the example subdirectory of the Windows NCAPI installation target directory.

    To run the example, open a cmd prompt, cd to the example directory, and execute the following command:

    ncs-googlenet-check.exe

    The GoogLeNet graph is not distributed with this example. You can build the GoogLeNet graph on a supported OS with the full NCSDK installed by running make compile in the ncsdk/examples/caffe/GoogLeNet directory. Provide the path to the compiled graph as a command line argument as follows:

    ncs-googlenet-check.exe -g path/to/graph

    Visual Studio Project

    An example visual studio project is located in ncsdk/examples/apps/win/ncs-googlenet-check_VS.

    The solution file is Format Version 12.00.

    Additional Examples

    For more examples, including Python examples, you can build the examples included with the ncsdk repository on a developer machine running a supported OS that has the full NCSDK installed. To build the examples, run make all in the ncsdk base directory. This will download all needed files and compile graph files. Copy the entire examples directory to your Windows machine.

    The C example binaries will need to be recompiled for Windows. You can use the included example Visual Studio project that is located in ncsdk/examples/apps/win/ncs-googlenet-check_VS as a template for creating Visual Studio solutions for C applications.

    You will need OpenCV to run the Python examples. One guide to setting up OpenCV with Python 3 on Windows 10 can be found here.

    Warning: The Makefiles are not set up for Windows. Rather than using the make run command, use a command like python run.py to execute the examples.

    Uninstallation

    To uninstall the NCAPI from your machine, search for it in Settings > “Add or Remove Programs” and click Uninstall.
